# **Gitbook 語法：連結**

---

在使用連結部份也有兩種方法可以進行

* 使用"[顯示文字](https://cowmanchiang.me/gitbook/gitbook/contents/how/%E9%80%A3%E7%B5%90)"進行

  ```
  [這是連結](http://cowmanchiang.me/gitbook)

  ```

  [這是連結](http://cowmanchiang.me/gitbook)


* 使用註解方式進行 \(示範語法中以 "【 】" 取代 "\[ \]" 以免發生誤判\)

  ```
  【這是連結】【1】

  【1】: http://cowmanchiang.me/gitbook
  ```
