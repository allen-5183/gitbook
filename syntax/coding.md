# **Gitbook 語法：程式碼**

---

在程式碼語法顯示部份也是有兩個方式可以進行標示

* 使用"8個空格"縮排

  ```
         ls -al

  ```

  ```
  ls -al

  ```


* 使用 " '''\[code language\] " \(以 " ''' " 取代 " \`\`\` " 避免排版錯亂\) 較推薦使用此標示方法，支援程式碼 highlight 顯示，支援哪些程式語言請參考 [highlightjs.org](https://highlightjs.org/static/demo/)

  ```
  '''bash
  ls -al
  '''

  ```

  ```
  ls -al
  ```
